# claudePA

Assistant personnel autonome de Sylvain Ribes, hébergé dans le cloud.
Tâches récurrentes de veille et de synthèse, mémoire en markdown
versionné, validation humaine systématique pour toute action externe.

## Documents clés

- **[`SPEC.md`](./SPEC.md)** — spec complète : architecture, stack,
  catalogue des tâches, mémoire à deux étages, validation humaine,
  garde-fous, phasage.
- **[`CLAUDE.md`](./CLAUDE.md)** — instructions pour les sessions Claude
  Code (handover permanent).
- **[`schedule.yaml`](./schedule.yaml)** — registre des tâches
  récurrentes (cron, canal, modèle).
- **[`core-memory/`](./core-memory/)** — mémoire permanente : profil,
  family, policies, plan Australie, réseau.
